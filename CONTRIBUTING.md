# Contributing to simai-env

Thanks for your interest in improving simai-env! Contributions are welcome. To keep things simple:

## How to contribute
- **Fork and branch**: create a feature branch from `main`.
- **Coding style**: keep scripts POSIX-friendly bash; prefer clear function names and comments only where logic is non-obvious.
- **Testing**: when possible, validate on Ubuntu 22.04/24.04; avoid breaking existing flags/flows.
- **Pull requests**: describe the change, rationale, and any manual steps to verify.

## Reporting issues
- Include OS version, command used (with flags), and relevant log snippets from `/var/log/simai-env.log`.
