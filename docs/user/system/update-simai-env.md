# Update Simai-Env

Use this item to update the installed `simai-env` scripts in place.

Choose it when:

- you intentionally want newer behavior or fixes,
- maintenance policy says the platform should be upgraded now.

What happens:

- the configured update target is fetched,
- a best-effort pre-update backup is created,
- a smoke check runs,
- menu sessions can reopen automatically after update.
