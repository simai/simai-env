Use `System` when the question is about the host or the platform layer itself, not about one application.

Recommended order:

1. `Platform status`
2. `Optimization status`
3. `Version` or `Check for updates now` when update state matters
4. `Update simai-env` or `Repair platform` only after a read-only check

Use another section instead when:

- only one site is affected: `Sites`, `Diagnostics`, or `Applications`
- you only need HTTPS state for one domain: `SSL`
