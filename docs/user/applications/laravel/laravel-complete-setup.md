# Laravel Complete Setup

Use this item to finalize a Laravel site after the application already exists.

What it typically does:

- final environment preparation,
- `APP_KEY` and storage baseline,
- scheduler baseline,
- optimization baseline,
- optional migrations and SSL choices when offered.

Choose it when:

- bootstrap is finished,
- the app code is already in place,
- you want the site to move from scaffold state to operational baseline.
