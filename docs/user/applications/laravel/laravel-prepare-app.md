# Laravel Prepare App

Use this item on a fresh Laravel-profile site that still has the placeholder structure.

Choose it when:

- the site was created with profile `laravel`,
- you want `simai-env` to bootstrap the base Laravel application shell and `.env` preparation.

What happens:

- a real Laravel app scaffold is prepared,
- environment wiring is staged for the site.

What to expect:

- this is an early lifecycle step,
- after it, continue with deployment or code sync if your project flow needs it,
- then run `Laravel complete setup`.
