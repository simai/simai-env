# Access

Use this section to give file access without handing out `root` or the main service account.

Menu items:

- [List accesses](./list-accesses.md)
- [Show access details](./show-access-details.md)
- [Create project access](./create-project-access.md)
- [Create global access](./create-global-access.md)
- [Add SSH key](./add-ssh-key.md)
- [Disable access](./disable-access.md)
- [Enable access](./enable-access.md)
- [Reset access password](./reset-access-password.md)
- [Remove access](./remove-access.md)

Typical flow:

1. `Create project access`
2. `Add SSH key`
3. `Show access details`
4. later use `Disable access` or `Remove access` when the work ends

- [access command reference](../../developer/commands/access.md)
