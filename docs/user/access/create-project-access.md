# Create Project Access

Use this item to create an SFTP-only account for one site.

Choose it when:

- a developer or integrator needs access to only one project,
- you want the safer default instead of wide server access.

What the wizard asks:

- domain,
- login,
- password or permission to autogenerate one.

What happens:

- the login is created as a managed SFTP-only account,
- access is limited to the selected project,
- the managed isolation path is prepared.

What to expect:

- this is the preferred option for ordinary project work,
- after creation, add a key if passwordless access is needed.
