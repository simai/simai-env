# Certificate Status

Use this item to inspect one domain's certificate in detail.

What it shows:

- certificate type,
- active cert and key paths,
- validity window,
- issuer,
- SAN names,
- staging flag and warnings.

Choose it when:

- HTTPS is broken for one site,
- you need to verify whether a custom or Let's Encrypt cert is active,
- you need a read-only check before renewal or replacement.
