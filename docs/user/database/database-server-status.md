# Database Server Status

Use this item to verify whether the database service itself is healthy.

What it tells you:

- service state,
- version,
- socket and port state,
- datadir health and free disk,
- basic reachability.

Choose it when:

- multiple sites report DB-related failures,
- `Prepare site database` or product setup cannot proceed,
- you need to decide whether the problem is the server or one site's credentials.
