# Site Info

Use this item as the main fact sheet for one site.

What it shows:

- domain, profile, project, slug,
- root and docroot,
- host mode and wildcard state,
- nginx and PHP pool details,
- SSL summary,
- scheduler and worker hints,
- activity class and optimization recommendation.

Use it when:

- you need to understand the current managed state before changing something,
- you want to know what the previous operator created,
- you need the next practical step after site creation.

What to expect:

- it is read-only,
- it does not repair anything,
- it is the best checkpoint before SSL, product setup, PHP switching, or removal.
