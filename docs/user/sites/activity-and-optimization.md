# Activity & Optimization

Use this item when you want a simple business-level view of how active the site is and how `simai-env` currently treats its runtime.

What it tells you:

- the current activity class,
- whether optimization is automatic, manual, or effectively paused,
- the recommendation for this site's footprint.

Choose it when:

- the server is under pressure and you need to see which sites are heavy,
- a site feels overprovisioned or underprovisioned,
- you want to know whether the next step should be `Change activity class`, `Pause site`, or an advanced optimization override.

What it does not do:

- it does not change settings,
- it does not pause traffic,
- it does not rewrite the PHP pool by itself.
