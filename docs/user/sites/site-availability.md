# Site Availability

Use this item to check whether the site's runtime is currently active or suspended.

What it answers:

- is the site serving normally,
- is the runtime parked,
- are related pool, cron, and queue pieces enabled or disabled.

Choose it when:

- a site owner says the project is unavailable,
- you are not sure whether someone intentionally paused the site,
- you need to decide between `Resume site` and deeper troubleshooting.

What to expect:

- it is read-only,
- it does not wake anything up,
- it is the safest checkpoint before pause or resume actions.
