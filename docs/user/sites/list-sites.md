# List Sites

Use this item when you need a compact inventory of all configured sites.

What happens:

- the menu runs the managed site listing,
- you see domain, profile, PHP version, root or alias target, and brief SSL state.

Use it when:

- you are not sure whether a site already exists,
- you need to confirm the profile before changing anything,
- you want a quick inventory before maintenance.

What to expect:

- this action does not change anything,
- it is safe to run at any time,
- it is usually the best first step before `Site info`, `Change site PHP`, or `Remove site`.
