# Daily Operations

Most everyday work falls into one of these patterns.

Check one site:

1. `Sites -> Site info`
2. `Diagnostics -> Site health check`
3. `SSL -> Certificate status`
4. `Applications -> <product> -> status`

Change runtime pressure for a site:

1. `Sites -> Activity & optimization`
2. `Sites -> Change activity class`
3. If the site should stop consuming resources for a while, `Sites -> Pause site`

Invite someone to work on files:

1. `Access -> Create project access`
2. `Access -> Add SSH key`
3. `Access -> Show access details`

Work with a certificate:

1. `SSL -> Certificate status`
2. `SSL -> Issue Let's Encrypt` or `Install custom certificate`
3. `Diagnostics -> Site health check`

Prepare before risky changes:

1. `Backup / Migrate -> Export site settings`
2. `Diagnostics -> Configuration check`
3. Only then do the change itself

Investigate a complaint:

1. `Diagnostics -> Site health check`
2. `Logs -> Website error log`
3. `Logs -> Platform log`
4. `Applications -> <product> -> status`
