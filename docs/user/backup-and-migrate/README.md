# Backup / Migrate

Use this section for safe export and import of managed site configuration archives.

Menu items:

- [Export site settings](./export-site-settings.md)
- [Review archive](./review-archive.md)
- [Preview import](./preview-import.md)

Recommended flow:

1. export,
2. inspect,
3. preview import,
4. only then run the actual import from Advanced mode if needed.

Use this section for managed config migration.
Do not treat it as a full content backup for project files, uploads, or secrets.
